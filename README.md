# TrendRadar - 全球金融市场分析

定时推送全球金融市场热点新闻，AI 智能筛选 + 全天汇总报告。

## 推送时间（北京时间）

| 时间 | 内容 |
|------|------|
| **07:50** | 盘前速览 |
| **09:10** | A股开盘跟进 |
| **12:50** | 午间热点 |
| **15:50** | A股收盘总结 |
| **21:20** | 美股开盘前瞻 |
| **23:50** | 🌟 全天最终报告（AI 深度汇总） |

## 配置

- **调度模式**: Custom（6 次/天 + 23:50 汇总）
- **AI 引擎**: Kimi K2.6 (Moonshot)
- **筛选方式**: AI 智能分类
- **数据源**: 华尔街见闻、财联社、Yahoo Finance、CNBC、MarketWatch

## 部署

GitHub Actions 定时触发，GitHub Pages 托管报告页面。
